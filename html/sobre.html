Visualizações Interativas da Segurança Pública

SOBRE A PÁGINA

Este Projeto foi desenvolvido como parte do Trabalho de Conclusão de Curso, referente ao Curso: Ciência da Computação da Universidade Federal de Campina Grande-PB.

O Projeto WEB foi desenvolvido como ferramenta para apresentar visualizações Interativas sobre dados referentes a Segurança Pública, mais específicamente da polícia Militar da Paraíba e Registros de Ocorrências da cidade de Campina Grande - PB, através das informações do Centro Integrado de Comando e Controle.
Tem como ênfase destacar a importância da Análise de Dados no setor Público, como forma de entender as informações geradas para analisar insight que possam auxiliar nas tomadas de decisões e gestão de recursos de forma mais eficiente.
Juntamente com a Análise de Dados temos a Visualização de Dados como recurso que podem contribuir significativamente para a Segurança Pública, tornando os dados brutos em informações mais claras e dinâmicas.
Nesse trabalho foi utilizado tecnologias como JavaScript, HTML, CSS, Bootstrap para exibir gráficos, mapas e análises visuais, além da biblioteca Leaflet para os mapas interativos, e também o Apache ECharts que é uma plataforma de visualização de dados baseada em JavaScript. Para o tratamento, preparação e análises dos dados foi utilizado a tecnologia Python e a biblioteca Pandas, deenvolvidos na plataforma Google Colab.
